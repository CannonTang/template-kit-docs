概览

TemplateKit

Unity 编辑器模板工具包

通过右键菜单快速创建预配置组件，提升开发效率。

动画

帧动画、简易动画器、路径动画

相机

相机控制、目标切换、视口偏移

网络通信

HTTP 数据源、WebSocket 客户端

UI 反馈

提示弹窗、顶部通知管理器

UI 控件

按钮组、开关、可拖拽面板、UI 跟随器

视觉运行时

圆点旋转加载等视觉效果

快速开始

在 Unity 编辑器 Hierarchy 面板中右键：

GameObject > 模板 > [分类] > [组件]

或通过 Tools 菜单生成预制体资源：

Tools > 模板 > 生成[分类]预制体资源

动画模块

Animation Kit

动画模块提供多种预配置的动画模板组件，涵盖帧动画、属性动画和路径动画。

GameObject > 模板 > 动画 > [选择组件]

帧动画控制器

动画 / FrameAnimator

截图 / GIF 占位建议尺寸: 800x450 或 GIF 演示

功能

播放序列帧动画，支持配置帧率、循环模式和播放控制。适用于 UI 图标动画、角色 2D 动画等场景。

使用方式

GameObject > 模板 > 动画 > 帧动画控制器

属性说明

属性类型说明
SpritesSprite[]帧动画序列图片
FrameRatefloat播放帧率
Loopbool是否循环播放

简易动画器 (位移旋转缩放)

动画 / SimpleAnimator

截图 / GIF 占位建议尺寸: 800x450

功能

对 Transform 属性进行简易动画控制，支持循环、Ping-Pong 等播放模式，无需 Animator Controller。

使用方式

GameObject > 模板 > 动画 > 简易动画器 (位移旋转缩放)

简易动画器 (图片填充)

动画 / ImageFillAnimator

截图 / GIF 占位建议尺寸: 800x450

功能

控制 Image 的 fillAmount 属性，实现进度条、冷却环、加载动画等填充效果。

使用方式

GameObject > 模板 > 动画 > 简易动画器 (图片填充)

路径动画器

动画 / PathAnimator

截图 / GIF 占位建议尺寸: 800x450

功能

沿预设路径点移动对象，支持 Scene 视图可视化编辑。适用于巡逻路线、摄像机轨道、UI 引导动画。

使用方式

GameObject > 模板 > 动画 > 路径动画器

相机模块

Camera Kit

相机模块提供相机控制、目标切换和视口偏移等功能模板。

GameObject > 模板 > 相机 > [选择组件]

相机控制器

相机 / CameraController

截图 / GIF 占位建议尺寸: 800x450

功能

提供相机跟随、缩放、旋转等基础控制，支持平滑过渡和边界约束。

相机目标切换器

相机 / CameraTargetSwitcher

截图 / GIF 占位建议尺寸: 800x450

功能

在多个目标间平滑切换相机关注点，支持过渡动画和缓动曲线配置。

视口偏移控制器

相机 / ViewportOffsetController

截图 / GIF 占位建议尺寸: 800x450

功能

控制相机视口偏移，适用于 UI 面板展开时需要偏移画面焦点的场景。

网络通信模块

Networking Core Kit

网络模块提供 HTTP 请求和 WebSocket 长连接的模板组件。

GameObject > 模板 > 网络通信 > [选择组件]

HTTP 数据源

网络通信 / HttpDataSource

截图 / GIF 占位建议尺寸: 800x450

功能

封装 HTTP 请求逻辑，支持 GET/POST，自动序列化/反序列化 JSON 数据。

HTTP 数据源管理器

网络通信 / HttpDataSourceManager

截图 / GIF 占位建议尺寸: 800x450

功能

统一管理多个 HTTP 数据源，提供全局配置和请求调度能力。

WebSocket 客户端

网络通信 / WebSocketClient

截图 / GIF 占位建议尺寸: 800x450

功能

WebSocket 长连接客户端，支持自动重连、心跳检测和消息收发。

UI 反馈模块

UI Feedback Kit

UI 反馈模块提供弹窗和通知的模板组件，开箱即用。

GameObject > 模板 > UI反馈 > [选择组件]

统一的提示弹窗管理，支持确认/取消双按钮、自定义内容和回调。Inspector 提供选项卡式配置面板。

  • 调试

截图 / GIF 占位建议尺寸: 800x450

功能

顶部滑入式通知消息，支持图标预设、自动消失和队列管理。

  • 图标预设

UI 控件模块

UI Controls Kit

UI 控件模块提供常用交互控件的模板，快速搭建界面交互。

GameObject > 模板 > UI控件 > [选择组件]

按钮组

UI 控件 / ButtonGroup

截图 / GIF 占位建议尺寸: 800x450

功能

管理一组互斥按钮（类似 Radio Group），自动处理选中状态切换。支持通过 Inspector 右键 Initialize 自动扫描子按钮。

开关按钮

UI 控件 / ToggleButton

截图 / GIF 占位建议尺寸: 800x450

功能

带状态切换的按钮组件，支持开/关两种视觉状态和回调。

可拖拽面板

UI 控件 / DraggablePanel

截图 / GIF 占位建议尺寸: 800x450

功能

支持拖拽移动的 UI 面板，可配置拖拽区域和移动约束。

UI 跟随器

UI 控件 / UIFollower

截图 / GIF 占位建议尺寸: 800x450

功能

使 UI 元素跟随指定目标（世界坐标或屏幕坐标），适用于血条、名称标签等。

UI 工具模块

UI Tools

编辑器辅助工具，提升 UI 开发效率。

对齐锚点到角落

UI 工具 / Anchors to Corners

截图 / GIF 占位建议尺寸: 800x450

功能

一键将 RectTransform 的锚点对齐到当前边界角落，解决 UI 自适应布局问题。

GameObject > 模板 > UI工具 > 对齐锚点到角落

视觉运行时模块

Visual Runtime Kit

运行时视觉效果组件模板。

圆点旋转加载

视觉运行时 / DotSpinner

截图 / GIF 占位建议尺寸: 800x450

功能

圆点旋转的 Loading 效果，纯代码实现，可自定义颜色、数量和大小。

GameObject > 模板 > 视觉 > 圆点旋转加载
